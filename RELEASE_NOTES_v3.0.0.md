# Release Notes v3.0.0

## English

- Introduced the 3.0.0 desktop workflow release, centered on project-based translation, guided onboarding, and a more complete Tauri desktop experience.
- Expanded the incremental translation pipeline with dedicated preparation, diff, snapshot, archive, build, package, and translation services.
- Upgraded the frontend with a fuller initial-translation flow, stronger project validation, richer activity/history views, and broader automated test coverage.
- Added backend and integration coverage for project management, system utilities, archive handling, proofreading, and workshop flows.
- Cleaned up the repository for public presentation by removing tracked temporary artifacts, replacing machine-specific launch paths with configurable scripts, and shipping a safe `.env.example`.

## 中文

- 发布 `3.0.0` 桌面工作流版本，围绕项目化翻译、内置引导和更完整的 Tauri 桌面体验进行了整体升级。
- 增量翻译链路完成拆分，新增 preparation、diff、snapshot、archive、build、package、translation 等独立服务模块。
- 前端完善了初始翻译流程、项目校验、活动/历史展示，并补充了更完整的自动化测试覆盖。
- 后端与集成测试覆盖范围扩展到项目管理、系统能力、归档处理、校对能力和 workshop 流程。
- 面向公开展示整理了仓库：移除已跟踪的临时产物，清理机器相关的硬编码启动路径，并提供安全的 `.env.example` 模板。
