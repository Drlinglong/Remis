# Archive

这里存放的是已经退出主入口、但仍保留为历史记录的文档。

这些文件通常属于以下类型：

- 旧版总览文档
- 阶段性报告
- 某次功能设计或方案讨论的长文

使用原则：

- 默认不要把这些文档当作当前项目事实。
- 需要理解某次历史决策、某轮演化背景或旧版本叙事时，再按需查阅。
- 如果归档文档与当前代码或根目录 `GEMINI.md` 冲突，以当前代码和 `GEMINI.md` 为准。

## 目录说明

- `legacy-overviews/`：旧版总览、旧版介绍页、重复入口
- `reports/`：阶段性报告、准备度报告、回顾文档
- `concepts/`：概念设计、方案探索、阶段性系统说明
- `developer-history/`：重构总结、集成总结、设计冻结文档等历史开发记录
